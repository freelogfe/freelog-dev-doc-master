**完成任务记录** <font color='blue'>**/v2/activities/task/records/complete4TaskConfigCode**</font>	**POST**	**JSON**

```
当指定条件的任务记录已存在时，则完成最高level的任务记录，若是该任务记录已完成，则抛出异常；不存在则先创建一个任务记录再完成，返回该任务记录
```

| -------------- | -------- | ------------ | ---- |

```
  }
}
```

| 参数名          | 参数类型 | 参数描述                         |
