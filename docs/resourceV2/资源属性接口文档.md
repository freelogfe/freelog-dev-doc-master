**创建资源属性** <font color='blue'>**/v2/resources/attrs/create**</font>	**POST**	**JSON**

| 参数名     | 参数类型 | 参数描述                                              | 必选 |
| ---------- | -------- | ----------------------------------------------------- | ---- |
| name       | string   | 名称                                                  | 是   |
| note       | string   | 备注                                                  | 否   |
| key        | string   | 键                                                    | 是   |
