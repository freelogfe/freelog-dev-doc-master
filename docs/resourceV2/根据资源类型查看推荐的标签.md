# 根据资源类型查看推荐的标签



### 调用方式: GET



### 接口地址:

```
url: https://api.freelog.com/v2/resources/tags/availableTags
```



### url传入参数说明:

| 参数 | 必选 | 类型及范围 | 说明 |
| :--- | :--- | :--- | :--- |
| resourceType | 可选 | string | 资源类型 |



### 返回说明：

| 返回值字段 | 字段类型 | 字段说明 |
| :--- | :--- | :--- |
| tagId | string | 标签ID |
| tagName | string | 标签名称 |
| tagType | string | 标签类型 1.分类标签 2.运营标签 |
