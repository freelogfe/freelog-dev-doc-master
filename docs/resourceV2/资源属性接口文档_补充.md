| 参数名           | 参数类型 | 参数描述     | 必选 |
| ---------------- | -------- | ------------ | ---- |
| name             | string   | 名称         | 是   |
| key              | string   | 键           | 是   |
| resourceTypeCode | string   | 资源类型编号 | 是   |
