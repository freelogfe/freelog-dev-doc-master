# 批量设置资源运营属性



### 调用方式: POST



### 接口地址:

```
https://api.freelog.com/v2/resources/operations
```



### body传入参数说明：

| 参数 | 必选 | 类型及范围 | 说明 |
| :--- | :--- | :--- | :--- |
|type | 必选 | int | 运营类型(1:编辑精选) |
|resourceIds | 必选 | string[] | 资源ID(已设置过的会被自动忽略) |



### body示例

```json
{
	"resourceIds": ["62e38b18793997002e07ed35", "62e386d0793997002e07eb14"]
}
```



### 返回说明：

| 返回值字段 | 字段类型 | 字段说明 |
| :--- | :--- | :--- |
| [data] | boolean | 操作结果 |



### 返回示例

```json
{
    "ret": 0,
    "errCode": 0,
    "errcode": 0,
    "msg": "success",
    "data": true
}
```
