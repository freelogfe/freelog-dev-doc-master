# 批量设置用户标签



### 调用方式: PUT



### 接口地址:

```
https://api.freelog.com/v2/users/batchSetTag
```



### body传入参数说明：

| 参数 | 必选 | 类型及范围 | 说明 |
| :--- | :--- | :--- | :--- |
| tagIds | 必选 | int[] | 标签ID |
| userIds | 必选 | int[] | 用户ID |



### body示例

```json
{
  "userIds":[50028,50022],
  "tagIds": [2,3]
}
```



### 返回说明：

| 返回值字段 | 字段类型 | 字段说明 |
| :--- | :--- | :--- |
| [data] | bool | 操作结果 |



### 返回示例

```json
{
    "ret": 0,
    "errcode": 0,
    "msg": "success",
    "data": true
}
```
