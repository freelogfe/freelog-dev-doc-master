# 取消用户标签



### 调用方式: PUT



### 接口地址:

```
https://api.freelog.com/v2/users/{userId}/unsetTag
```



### url传入参数说明：

| 参数 | 必选 | 类型及范围 | 说明 |
| :--- | :--- | :--- | :--- |
|userId|必选|int|用户ID|



### body传入参数说明：

| 参数 | 必选 | 类型及范围 | 说明 |
| :--- | :--- | :--- | :--- |
|tagIds|必选|int[]|tagId|



### body示例

```json
{
  "tagIds":[15]
}
```



### 返回说明：

| 返回值字段 | 字段类型 | 字段说明 |
| :--- | :--- | :--- |
| [data] | bool | 操作结果 |



### 返回示例

```json
{
    "ret": 0,
    "errcode": 0,
    "msg": "success",
    "data": true
}
```
