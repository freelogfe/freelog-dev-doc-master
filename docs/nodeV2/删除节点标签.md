# 批量删除节点标签 (指删除标签库里的标签)



### 调用方式: DELETE



### 接口地址:

```
https://api.freelog.com/v2/nodes/tags
```



### url传入参数说明：

| 参数 | 必选 | 类型及范围 | 说明 |
| :--- | :--- | :--- | :--- |
| tagIds | 必选 | string | 标签ID,多个用逗号分隔 |



### 返回说明：

| 返回值字段 | 字段类型 | 字段说明 |
| :--- | :--- | :--- |
| [data] | bool | 操作结果 |



### 返回示例

```json
{
    "ret": 0,
    "errcode": 0,
    "msg": "success",
    "data": true
}
```
