# 批量设置或移除节点标签



### 调用方式: PUT



### 接口地址:

```
https://api.freelog.com/v2/nodes/batchSetOrRemoveNodeTag
```



### body传入参数说明：

| 参数 | 必选 | 类型及范围 | 说明 |
| :--- | :--- | :--- | :--- |
|tagNames | 必选 | string[] | 标签 |
|nodeIds | 必选 | int[] | 节点ID |
|setType | 必选 | int | 操作类型 1:设置标签  2:移除标签 |



### body示例

```json
{
  "tagNames":["tag1","tag2"],
  "nodeIds":[80000053,80000055],
  "setType":2
}
```



### 返回说明：

| 返回值字段 | 字段类型 | 字段说明 |
| :--- | :--- | :--- |
| [data] | bool | 是否操作成功 |



### 返回示例

```json
{
    "ret": 0,
    "errCode": 0,
    "errcode": 0,
    "msg": "success",
    "data": true
}
```
