# 创建节点标签



### 调用方式: POST



### 接口地址:

```
https://api.freelog.com/v2/nodes/tags
```



### body传入参数说明：

| 参数 | 必选 | 类型及范围 | 说明 |
| :--- | :--- | :--- | :--- |
|tags | 必选 | string[] | tag名称 |



### body示例

```json
{
  "tags":["节点标签4","节点标签5"]
}
```



### 返回说明：

| 返回值字段 | 字段类型 | 字段说明 |
| :--- | :--- | :--- |
| tagId | int | 标签ID |
| tag | string | 标签名称 |
| totalSetCount | int | 总设置次数 |
