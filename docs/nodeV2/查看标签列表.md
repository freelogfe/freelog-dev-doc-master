# 查看节点标签列表



### 调用方式: GET



### 接口地址:

```
https://api.freelog.com/v2/nodes/tags
```



### 返回说明：

| 返回值字段 | 字段类型 | 字段说明 |
| :--- | :--- | :--- |
| tagId | string | 标签ID |
| tagName | string | 标签名称 |
