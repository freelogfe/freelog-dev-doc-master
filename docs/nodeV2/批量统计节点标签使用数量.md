# 批量统计节点标签使用数量



### 调用方式: GET



### 接口地址:

```
https://api.freelog.com/v2/nodes/tags/statistics
```



### url传入参数说明：

| 参数 | 必选 | 类型及范围 | 说明 |
| :--- | :--- | :--- | :--- |
|tagIds | 必选 | string | 标签ID,多个用逗号分隔 |



### 返回说明：

| 返回值字段 | 字段类型 | 字段说明 |
| :--- | :--- | :--- |
| tagId | string | 标签ID |
| tagName | string | 标签 |
| count | int | 标签使用数量 |



### 返回示例

```json
{
	"ret": 0,
	"errCode": 0,
	"errcode": 0,
	"msg": "success",
	"data": [{
		"tagId": "612c8cf1f7e09660947df8e1",
		"tagName": "tag1",
		"count": 1
	}]
}
```
