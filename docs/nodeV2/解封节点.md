# 解封节点



### 调用方式: PUT



### 接口地址:

```
https://api.freelog.com/v2/nodes/{nodeId}/deArchive
```



### url传入参数说明：

| 参数   | 必选 | 类型及范围 | 说明   |
| :----- | :--- | :--------- | :----- |
| nodeId | 必选 | int        | 节点ID |



### body传入参数说明：

| 参数   | 必选 | 类型及范围 | 说明                   |
| :----- | :--- | :--------- | :--------------------- |
| remark | 可选 | string     | 备注信息   |
| reason | 可选 | string | 原因 |



### body示例

```json
{
    "remark": "解封"
}
```



### 返回说明：

| 返回值字段 | 字段类型 | 字段说明 |
| :--------- | :------- | :------- |
| [data]     | bool     | 操作结果 |



### 返回示例

```json
{
    "ret": 0,
    "errcode": 0,
    "msg": "success",
    "data": true
}
```
