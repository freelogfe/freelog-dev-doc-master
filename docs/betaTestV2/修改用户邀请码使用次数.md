# 修改用户邀请码使用次数

> 内部服务端具备权限,外网无调用权限
>



### 调用方式: PUT



### 接口地址:

```
https://api.freelog.com/v2/testQualifications/beta/codes/limitCount
```



### body传入参数说明：

| 参数 | 必选 | 类型及范围 | 说明 |
| :--- | :--- | :--- | :--- |
| userId | 必选 | int | 用户ID |
| incrNumber | 必选 | int | 新增或减少的次数, 正数为新增,负数为减少 |



### body示例

```json
{
	"userId": 50017,
	"incrNumber": 2
}
```



### 返回说明：

| 返回值字段 | 字段类型 | 字段说明 |
| :--- | :--- | :--- |
| [data] | boolean | 是否操作成功 |



### 返回示例

```json
{
	"ret": 0,
	"errcode": 0,
	"msg": "success",
	"data": true
}
```
