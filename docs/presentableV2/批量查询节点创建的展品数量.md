# 批量查询节点创建的展品数量(for admin)



### 调用方式: GET



### 接口地址:

```
https://api.freelog.com/v2/presentables/admin/presentableStatistics
```



### url传入参数说明：

| 参数 | 必选 | 类型及范围 | 说明 |
| :--- | :--- | :--- | :--- |
| nodeIds | 必选 | string | 节点ID,多个用逗号分隔 |



### 返回说明：

| 返回值字段 | 字段类型 | 字段说明 |
| :--- | :--- | :--- |
| nodeId | int| 节点ID |
| count | int | 创建展品数 |



### 返回示例

```json
{
	"ret": 0,
	"errCode": 0,
	"errcode": 0,
	"msg": "success",
	"data": [{
		"nodeId": 80000004,
		"count": 4
	}, {
		"nodeId": 80000000,
		"count": 70
	}, {
		"nodeId": 80000088,
		"count": 0
	}]
}
```
